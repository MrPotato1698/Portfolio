## 🚀 Proyectos Destacados

### ESP Racing Web Application
Aplicación web para la comunidad ESP Racing con información de carreras, estadísticas e inscripciones.
- **Stack**: Astro, TailwindCSS, TypeScript, Node.js, Supabase, Vercel
- **Link**: https://esp-racing.com/

### Juego del Impostor
Juego interactivo desarrollado en TypeScript, que se convirtió en un clásico para las reuniones.
- **Stack**: TypeScript, TailwindCSS
- **Repository**: https://github.com/MrPotato1698/ProyectoImpostor

### Simulador DMN
Pequeño simulador de motor de reglas de negocio (DMN) desarrollado para resolver problemas de gestión de proyectos.
- **Stack**: TypeScript, TailwindCSS
- **Repository**: https://github.com/MrPotato1698/Modeler-DMN-Propio

## 🛠️ Comandos Disponibles

```bash
# Instalar dependencias
pnpm install

pnpm dev

# Generar build de producción
pnpm build

# Previsualizar build
pnpm preview
```
